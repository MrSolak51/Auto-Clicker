![clicked_key-ezgif com-crop](https://github.com/user-attachments/assets/a51a3e28-ec74-476e-8f27-3d9013e658b2)# Auto-Clicker
A program that simulates mouse and keyboard events. Created using C with GTK and Windows libraries.

USAGE:
- Select the key that will initiate the click event.
  ![binding-ezgif com-crop](https://github.com/user-attachments/assets/8c2fe34b-198c-41ac-a7b4-b3e19e11bede)
- Select the CPS (Clicks per second) level.
  ![CPS-ezgif com-crop](https://github.com/user-attachments/assets/6cb985a0-78c3-485e-841b-7ccbf6f87af4)
- Select the key on which the click event will occur.
  ![clicked_key-ezgif com-crop](https://github.com/user-attachments/assets/b38c3a90-ef23-42c7-b23e-1c8d317e6ff2)
- When you click the Bind key, the click event will start.
  ![clicking-ezgif com-crop](https://github.com/user-attachments/assets/845c7168-d294-4017-90d0-55a9966c0c6f)
